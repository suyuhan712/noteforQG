# 为什么说java是面向对象的编程

C语言采用"以过程为中心"的编程范式，强调将问题拆解为一系列具体步骤，通过函数逐步实现并依次调用

java需要先定义类，再创建对象

封装：数据不能随便改，必须通过方法修改


继承：让一个类拥有另一个类的属性和方法。

多态：同一个方法，不同对象表现不同

面向接口编程，而不是面向实现编程。（不需要大改代码）

用户访问网站：
用户输入URL
      ↓
DNS解析
      ↓
TCP三次握手
      ↓
HTTP请求
      ↓
服务器处理
      ↓
HTTP响应
      ↓
浏览器渲染
